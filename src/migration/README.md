# Trigger Migration Layout

这个目录保存从旧 JASS/GUI 触发器迁移到 TypeScript 的实现。

- `index.ts`: 迁移总入口，按地图启动顺序注册各分类触发器。
- `core/`: 迁移公共工具、旧触发器禁用/替换、遗留触发器接管。
- `flow/`: 开局、难度、昼夜、波次、Boss 流程等全局流程。
- `items/`: 物品获取、合成、装备、商人、传送卷轴等物品相关触发器。
- `combat/`: 技能、战斗、伤害、敌人死亡、职业/怪物能力等战斗相关触发器。
- `objectives/`: 任务、得分、胜负、资源显示等目标/计分相关触发器。
- `heroes/`: 英雄选择相关触发器。
- `systems/`: 镜头、调试、复活等辅助系统触发器。
- `groups/`: 触发器分组元信息。
